# SafePath Browser Demo — GitHub Pages Deploy

## Quick Deploy (no coding)
1. Create a new GitHub repo (public) named: safepath-browser (or any name).
2. Upload ALL files from this folder to the root of the repo (index.html must be at repo root).
3. In GitHub: Settings → Pages → Source: Deploy from a branch → Branch: main / root → Save.
4. Your site will be live at:
   https://YOURUSERNAME.github.io/REPO_NAME/

## Notes
- Open two tabs to simulate User + Guardian mode.
- The app is 100% static; no server required.
